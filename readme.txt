== Description ==

This plugin allows the adminstrator to reorder or rearrange wordPress dashboard admin panel menus & submenus.

If you need additional help you can contact us for [Custom Development](https://wbcomdesigns.com/contact/).

== Installation ==

1. Upload the entire "dashboard-menu-editor" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
